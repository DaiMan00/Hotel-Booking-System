This C++ script is a hotel booking system that allows the user to manage the resources owned by a hotel.

Firstly, the user has to set the total number of hotel rooms, pastas, pizzas.

Secondly, the user has to select how many hotel rooms, pastas, pizzas are sold in the menu.

Thirdly, the user can check the number of items that are sold, and the quantity of each item that is still available. Also, the user can check the revenues of each item and total sales revenues.
